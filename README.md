## 📍 Overview

AdminBase is a clean and reusable PHP boilerplate built for authenticated admin systems. It includes session-based login, protected routes, reusable layout components, and a generic dashboard structure designed to be adapted into internal tools, CRUD systems, admin panels, and other back-office applications.

## 👾 Features

* Session-based authentication with login and logout flow
* Protected routes for private pages
* Reusable admin dashboard layout
* Generic structure ready for future modules and CRUDs
* Clean folder organization with separated config, core logic, pages, includes, and assets
* Responsive interface built with PHP, CSS, and JavaScript only

## 🚀 Getting Started

### ☑️ Prerequisites

Before getting started with adminbase, ensure your runtime environment meets the following requirements:

* Programming Language: PHP 8.0 or higher
* Local Server: XAMPP, Laragon, WAMP, MAMP, or PHP built-in server

### ⚙️ Installation

1. Clone the adminbase repository:

git clone [https://github.com/iLevisz/adminbase.git](https://github.com/iLevisz/adminbase.git)

2. Navigate to the project directory:

cd adminbase

3. Start a local PHP server:

php -S localhost:8000

### 🤖 Usage

Open the project in your browser:

[http://localhost:8000](http://localhost:8000)
